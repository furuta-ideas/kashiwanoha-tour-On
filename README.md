# kashiwanoha-tour-On

柏の葉スマートシティツアー「＋Onツアー」のランディングページです。

- 公開URL: https://furuta-ideas.github.io/kashiwanoha-tour-On/
- 構成: 静的サイト（`index.html` + `assets/lp/`）。ビルド不要で、GitHub Pages からそのまま配信されます。

## 主な機能

- 視察プラン6コースの詳細モーダル
- お問い合わせフォーム（入力内容から件名・本文入りの mailto リンクを生成）
- 実績数値のカウントアップ、スクロール連動のフェードイン表示（`prefers-reduced-motion` 対応）
- FAQ アコーディオン、モバイル対応（ハンバーガーメニュー）
