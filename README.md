# kashiwanoha-tour-On

柏の葉スマートシティ「＋Onツアー」（柏の葉スマートシティツアーズ）のランディングページです。

- 公開URL: https://furuta-ideas.github.io/kashiwanoha-tour-On/
- 構成: 静的サイト（`index.html` + `assets/lp/`）。ビルド不要で、GitHub Pages（main ブランチ / ルート）からそのまま配信されます。

## 主なセクション

- ファーストビュー（実績カウントアップ 45回 / 701名）
- ＋Onツアーとは（公式ツアーとの比較表）
- こんな方におすすめ／得られること／ツアーの流れ（6ステップ）
- 追加オプション（AIワークショップ・クイズ・成果物整理などの実例画像付き）
- 実績・掲載紹介（メディアリンク）／お客様の声（順次掲載）
- 代表者紹介（古田邦夫・関連記事リンク）／FAQ（10問）／お問い合わせCTA

## SEO

- title / description / keywords、canonical、OGP / Twitter カード
- JSON-LD 構造化データ（Organization / Person / TouristTrip / FAQPage）
- `robots.txt` / `sitemap.xml`

## 更新方法

`index.html` を編集して main に push すると、数十秒〜数分で公開URLに反映されます。
画像は `assets/lp/` に配置してください。
